# Contributing

Thanks for helping improve DazeStack WP.

## How to contribute
1. Fork the repo and create a feature branch.
2. Keep changes focused and explain the why.
3. Update documentation when behavior changes.
4. Submit a pull request with a clear summary and test notes.

## Development tips
- This is a Bash script targeting Ubuntu 24.04+.
- Prefer POSIX-safe constructs when possible.
- Keep logs and output ASCII-safe for wide terminal compatibility.
